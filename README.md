# SonicGlow - Premium Audio E-commerce

SonicGlow is a modern, responsive e-commerce website concept for high-end audio gear. It features a sleek dark-themed design with gold accents, providing a premium user experience.

![SonicGlow Preview](https://placehold.co/1000x500/0f0f0f/d4af37?text=SonicGlow+Preview)

## Features

-   **Modern UI/UX**: Dark mode aesthetic with smooth animations and transitions.
-   **Responsive Design**: Fully functional on Desktop, Tablet, and Mobile devices.
-   **Product Catalog**: Grid layout displaying product details and images.
-   **Dynamic Shopping Cart**:
    -   Add items to cart.
    -   Real-time quantity updates.
    -   Automatic total price calculation.
    -   Slide-out sidebar interface.
-   **Multiple Pages**:
    -   **Home**: Hero section, Product Collection.
    -   **About**: Brand story and values.
    -   **Contact**: Functional contact form layout and information.

## Technologies Used

-   **HTML5**: Semantic structure.
-   **CSS3**: Custom styling, Flexbox, Grid, CSS Variables, and Animations.
-   **JavaScript (Vanilla)**: DOM manipulation, Cart logic, and Event handling.
-   **Ionicons**: For scalable vector icons.
-   **Google Fonts**: Typography (Outfit).

## Usage

1.  Clone the repository:
    ```bash
    git clone https://github.com/NXRts/Audio-Ecommerce-.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd Audio-Ecommerce-
    ```
3.  Open `index.html` in your web browser.

## Customization

-   To modify product data, edit the `products` array in `script.js`.
-   To change specific colors or fonts, update the CSS variables in the `:root` section of `style.css`.
